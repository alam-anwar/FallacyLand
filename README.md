# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## To install and run this website on your computer:

1. Ensure you have node and npm installed.
2. `cd` into the project directory and run `npm install`. This will install all necessary files and binaries.
3. Still in the project directory. run `npm run dev`.
4. Navigate to http://localhost:5173/.
